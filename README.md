# Java Bank Account Manager

## Overview

**Java Bank Account Manager** is a simple console-based Java application that allows users to manage bank accounts. The program provides core functionality such as creating accounts, depositing and withdrawing funds, checking balances, and viewing account details.  

It is designed as a learning project to demonstrate basic Java programming concepts, including:

- Object-oriented programming (classes and objects)
- File handling
- Basic input/output
- Error handling and validation

---

## Features

- **Create a bank account** – Users can open new accounts with a unique account number.
- **Deposit funds** – Add money to an account.
- **Withdraw funds** – Remove money from an account, with validation to prevent overdrawing.
- **Check balance** – Display the current balance of an account.
- **View account details** – Show account number, owner name, and balance.
- **Persistent storage** – Accounts can optionally be saved to and loaded from files (if implemented).

---

## Prerequisites

- Java Development Kit (JDK) 8 or later installed
- IDE (optional, e.g., IntelliJ IDEA, Eclipse, or VS Code with Java extensions)
- Basic understanding of running Java programs from terminal or IDE
